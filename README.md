# ProjetPOO_Java
Projet Java MVC : gestion d'un magasin d'instruments (vente, réparation, location, accessoires).

---


Description du projet

Ce projet a été réalisé dans le cadre du cours de Programmation Orientée Objet (Java).
L’objectif était de modéliser et gérer un magasin proposant plusieurs services :

• Vente d’instruments
• Location
• Réparation
• Achat d’accessoires


Le programme permet de gérer les différents éléments du magasin via une interface graphique Java (Swing) en appliquant les principes de la POO : classes, héritage, contrôleurs, vues.

---

Fonctionnalités principales

• Gestion des instruments : ajout, suppression, modification, consultation
• Gestion des locations : louer, rendre, vérifier disponibilité
• Gestion des réparations : enregistrer une demande
• Gestion des accessoires : stock, prix, disponibilité
• Interface graphique : vues dédiées, contrôleur central
• Connexion gérant : accès aux fonctionnalité de gestion du magasin


---

---

Architecture du projet

• Modèle : classes représentant les instruments, accessoires, locations, réparations…
• Vue : interfaces graphiques permettant l’interaction avec l’utilisateur
• Contrôleur : gestion des événements, communication entre modèle et vue


Ce découpage permet une meilleure organisation du code.

---

Exécution du programme

1. Cloner le projet
2. Ouvrir dans un IDE Java (VS Code, IntelliJ, Eclipse...)
3. Compiler et exécuter la classe principale (Main.java)
4. Naviguer dans l’interface
5. Se connecter en tant que gérant pour accéder aux fonctionnalités avancées
